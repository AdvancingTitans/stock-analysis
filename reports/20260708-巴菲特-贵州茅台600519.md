## 三、商业质量与护城河

### 1. 这门生意十年后是否仍强？

高端白酒的需求与社交/收藏/礼品属性绑定，茅台品牌在国内心智中仍处顶层。只要 **品牌不被稀释、产能与品质纪律不乱、渠道不系统性窜货崩盘**，十年可理解性仍然成立。

### 2. 护城河来源

| 来源 | 证据方向 | 状态 |
|---|---|---|
| 品牌 | 长期溢价与高毛利 | 财务支持 |
| 产能/品质稀缺 | 飞天等核心 SKU 供给约束 | 定性，未在本包量化 |
| 渠道网络 | 经销体系与直营比例变化 | **本包缺口** |
| 监管/产区 | 地理与工艺壁垒 | 定性 |

### 3. 红旗检查（巴菲特红线）

| 红旗 | 是否触发 |
|---|---|
| 依赖短期融资讲故事 | 否（低杠杆） |
| 利润好看但自由现金流长期跟不上 | 目前 **未触发**（FCF-lite 高） |
| 高杠杆 + 周期下行 | 否 |
| 管理层只讲故事、资本回报下降 | 需跟踪 ROE 趋势：2025FY 32.53% vs 2024FY 36.02%，**略降需持续观察** |

### 巴菲特式解读

1. **毛利率 ~90%**：极少数企业具备的定价权证据。  
2. **杠杆极低**（资产负债率约 12–16%）：永久性损失风险显著低于周期股/高杠杆金融。  
3. **现金流**：2025FY 经营现金流 615 亿、FCF-lite 584 亿；2026Q1 单季 FCF-lite 263 亿，显示现金创造能力仍在。  
4. **增长放缓信号**：2025FY 营收/净利略低于 2024FY；2026Q1 营收同比约 +6.3%（547/514），净利小幅增长。属于 **高质量但增速换挡**，不是崩塌。  
5. **EPS / BPS**：2025FY 基本 EPS 65.66、BPS 195.36；2026Q1 EPS 21.76、BPS 216.32。

## 七、风险（永久性本金损失视角）

1. **需求与批价走弱**导致护城河“体感”下降（未在本包直接量化）  
2. **增长换挡**使市场从“成长溢价”切换到“现金牛折现”，估值中枢下移  
3. **政策与消费税**预期变化  
4. **集中持仓风险**：若组合过度集中茅台，单一公司事件冲击放大  

未发现：高杠杆爆仓、现金流断裂、会计红旗（在已披露摘要层面）。
