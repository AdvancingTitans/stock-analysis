# 持仓分析模板

持仓分析必须按以下顺序原样套用三个 Markdown 表格。单个字段缺失时保留空单元格，不得改写成段落。

## 表 1：持仓明细

| 代码 | 名称 | 市场 | 买入日 | 数量 | 现价 | 当日涨跌 | 当日浮盈/亏 | 趋势 |
|---|---|---|---|---:|---:|---:|---:|---|
|  |  |  |  |  |  |  |  |  |

## 表 2：组合概况与风险

| 总市值(CNY) | 总浮盈/亏 | 前三大占比 | 单一市场最高暴露 | 风格暴露 |
|---:|---:|---:|---|---|
|  |  |  |  |  |

## 表 3：基准相对强弱

| 代码 | 名称 | 基准指数 | 跑赢/跑输(pp) |
|---|---|---|---:|
|  |  |  |  |

规则：

- 三个表格均必须输出，即使暂无持仓或部分字段缺失。
- 现价保留原始币种；总市值和总浮盈/亏统一折算为 CNY。
- 跑赢为正数、跑输为负数，单位为百分点（pp）。
- 趋势使用已有 MA5/MA10/MA20 结论，不得根据缺失数据自行推断。
