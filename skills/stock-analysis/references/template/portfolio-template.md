# 持仓分析模板

持仓分析必须由用户主动触发。只有股票代码、买入日期、买入数量或买入金额三项完整时，才按以下顺序原样套用三个 Markdown 表格。单个字段缺失时保留空单元格，不得改写成段落。

如果用户没有主动提供持仓信息，或主动提供但缺少任意核心项，输出普通市场复盘报告，不套用本模板，不进行收益计算。信息不完整时只提问一次，精准指出缺失项；若用户只给出数字，必须确认这是买入数量还是买入金额，并确认币种是人民币、港币还是美元。

规则：

- 三个表格仅在用户主动触发且完整持仓信息成立时输出。
- 完整性口径固定为：股票代码、买入日期、买入数量或买入金额。
- 缺失任意一项，不得进行收益计算、浮盈亏计算、年化表现或组合绩效分析。
- 日期没有年份时默认使用当前年份；完整年份必须严格按用户提供值。
- 现价保留原始币种；总市值和总浮盈/亏统一折算为 CNY。
- 跑赢为正数、跑输为负数，单位为百分点（pp）。
- 趋势使用已有 MA5/MA10/MA20 结论，不得根据缺失数据自行推断。
