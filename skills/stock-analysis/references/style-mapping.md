# 研报术语映射

正式报告只呈现市场事实、分析结论和风险，不呈现工程实现。

| 工程侧原始表述 | 研报标准表述 |
|---|---|
| `aftermarket.py` 脚本采集到 | 据公开市场数据 |
| 早盘/实时采集 | 盘中数据显示 |
| 推测为全天数据 | 按惯例回溯至该日 |
| 存在不确定性 | 数据口径有待确认 |
| M1/M2/M3 降级 | 本模块证据暂缺 |
| fallback 到 | 切换至相应公开市场数据 |
| 被限流/被封 | 数据源暂未更新 |
| HTTP 错误/异常堆栈 | 该指标当日未披露 |

输出前必须扫描并移除：

- 文件扩展名：`.py`、`.js`、`.sh`
- 本地路径
- 实现关键词：`aftermarket`、`push2`、`exa`
- 工程措辞：脚本、采集、推测、不确定性、猜测、降级、fallback
